# Examples

This directory contains demo scripts and validation tools for ODAVL Guardian.

## Demo Scripts

- `demo-delta-insight.js` - Demonstrates delta insights functionality
- `demo-focus-summary.js` - Shows focus summary features
- `demo-pattern-detection.js` - Pattern detection examples
- `demo-pattern-synthetic.js` - Synthetic pattern generation
- `demo-silence-discipline.js` - Silence discipline demonstration

## Validation & Testing

- `validate-stage-v.js` - Stage V validation script
- `stage-v-summary.js` - Stage V summary generation
- `test-integration-patterns.js` - Integration pattern testing

## Usage

Run any demo script with Node.js:

```bash
node examples/demo-pattern-detection.js
```

These scripts are for development and demonstration purposes only and are not part of the main Guardian CLI.
